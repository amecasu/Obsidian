# FMEA should be...
1. CLEAR
	- 오역이 없도록 정확하게 작성한다.
	- 위험 수준에 대해 감정적인 평가를 하지 않는다.
2. TRUTHFUL
	- 발생할 수 있는 불량을 경시하지 않는다. 복잡한 뒤처리가 따르더라도,
3. CLEAR
	- 무결성. 완전해야 한다.

모두가 봐도 알아볼 수 있도록 쉽고 명확하게 적어야 함.

### 언제 업데이트 해야 하는가?
- 공정이나 제품이 변경될 때,
- 작업조건이 변경될 때,
- 필요 요구사항이 변경된 경우(고객사, 관련 법규, 표준 등등)
- 0마일리지, 플랜트 내부에서 발견된 즉각적인 불량이 있는 경우
- Negative finding이 발견될 경우
